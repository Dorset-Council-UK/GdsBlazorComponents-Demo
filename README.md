# GDS Blazor Components Demo

A demonstration application showcasing the [Dorset Council GDS Blazor Components](https://github.com/Dorset-Council-UK/GdsBlazorComponents) library.

## Overview

This Blazor Server application demonstrates all major components from the GDS Blazor Components library, which implements the GOV.UK Design System in Blazor.

## Running Locally

### Prerequisites
- .NET 10 SDK

### Steps
1. Clone the repository
2. Navigate to the project directory
3. Run `dotnet run`
4. Open your browser to the URL displayed (usually https://localhost:5001)

## Notes
This demo application is only intended to show the features available in the GdsBlazorComponents library. 
It is not intended as a best practices guide for using the library, or using GDS components and styles in your project.

This demo is not run by or any way affiliated with the UK Government or the GOV.UK Design System team. 
It is an independent project created by Dorset Council to demonstrate the GDS Blazor Components library.
